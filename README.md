- Use manipulators in a program to format output
- Perform input and output operations using the string data type

## Note on Coverage

- **Individual tutorials (1–13)** focus on fundamentals: setup, variables, basic I/O (cout, cin), and functions.
- **Control structures, loops, enums, namespaces, and advanced string/file I/O** are covered in the **All-in-One** video. Use the timestamp links to jump to those sections.
- **Input stream functions (get, ignore, putback, peek)** are not clearly titled in these videos; check the All-in-One around the string and I/O sections, or supplement with C++ reference material.
- **Manipulators** for output formatting are typically introduced alongside cout; review Tutorial 7 and the All-in-One output sections.
- **Drivers and stubs** for debugging are a testing technique; the All-in-One includes "Refactoring IO to Function Call and Testing" (6:47:53), which illustrates similar practices.
